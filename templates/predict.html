About This AI Model

FraudShield AI uses a Random Forest Machine Learning model trained on the popular Kaggle Credit Card Fraud Detection dataset.


The model analyzes transaction patterns based on Transaction Time and Transaction Amount to classify transactions as either fraudulent or normal.


The prediction shown above includes the confidence score generated by the machine learning model.

⚡ Fast Prediction

Predicts fraud within milliseconds using a pre-trained Random Forest model.

🛡 Fraud Detection

Detects suspicious credit card transactions using machine learning instead of manual rules.

📊 Dataset

Trained using 284,807 real-world credit card transactions from the Kaggle Fraud Detection dataset.

🤖 AI Powered

Powered by Scikit-Learn Random Forest Classifier with saved model deployment using Joblib.

💾 Backend

Developed using Python, Flask and MySQL with a responsive banking dashboard interface.

📈 Enterprise Ready

Designed to demonstrate how AI can assist banks in identifying potentially fraudulent transactions.
